Detailed Description

A generic algorithm based on a value function.

This class takes care of storing and loading a V function from a file, and can be sub-classed to implement update rules.

The value of the states are either initialised from the aforementioned file or set to defaultV when the value is requested for the first time. A value is not stored until the corresponding state is visited.
