Detailed Description

The base class for parser-based condition checkers.

This class uses a flex-bison parser to compute the actual value of a condition. Atomic condition checking is delegated to subclasses through the method ::evaluateAtomicCondition(const std::string&) .

The grammar of conditions is as follows:

 condition -> '(' (and|or) condition condition+ ')' |
              '(' not condition ')'                 |
              ATOMIC_CONDITION

where "and", "or" and "not" are case insensitive.
